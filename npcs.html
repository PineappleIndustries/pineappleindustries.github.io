N.P.C.'s

Last updated: June 13th, 2022

What It Is

The N.P.C.'s plugin (or "Citizens") allows you to create custom N.P.C.'s that are able to walk around, talk to players, and much more. Here is a guide on some commands that I used to create N.P.C.'s.
Anything that you should change in the commands will be highlighted in pink and underlined, like so.

Working with N.P.C's

Creating an N.P.C.

To create an N.P.C, type "/npc create [npc-name]". An example of this would be "/npc create John." This should spawn an npc where you are with the name you specified above its head. Other than that, there is not much that it does. We should be able to change that with a few more commands.

Altering the N.P.C.'s Appearance

I believe that the N.P.C. that you just created should be selected, but if any commands that you run to modify it show up on another N.P.C. or do not work and show up with a message similar to "No N.P.C. selected," look at the N.P.C. you created and type "/npc select. This should select the N.P.C.

To have the N.P.C. stare or look at players who approach it, type "/npc lookclose". It should now begin to look at you.

To alter the mob type of the N.P.C. (from a player to something else if you have not changed its mob type yet), type "/npc type [mob-type]". An example of this would be "/npc type pig."

To alter the skin of the N.P.C., what I have been using is "/npc skin [username]". An example of this would be "/npc skin Technoblade," which would alter the N.P.C.'s skin to the skin of the account with the username Technoblade.

Adding Dialogue (or Monologue) to N.P.C.'s

Working with the N.P.C. text editor is able to be a bit tricky, from my experience (you may not experience that, though). You will open up the N.P.C. text editor by typing "/npc text". A message in chat should notify you that you have opened the N.P.C. text editor. You should be provided with a few options on what you would like to do, but I will explain them here just for the sake of clarity.
