N.P.C.'s

Last updated: June 13th, 2022

What It Is

The N.P.C.'s plugin (or "Citizens") allows you to create custom N.P.C.'s that are able to walk around, talk to players, and much more. Here is a guide on some commands that I used to create N.P.C.'s.
Anything that you should change in the commands will be highlighted in pink and underlined, like so.

Working with N.P.C's

Creating an N.P.C.

To create an N.P.C, type "/npc create [npc-name]". An example of this would be "/npc create John." This should spawn an npc where you are with the name you specified above its head. Other than that, there is not much that it does. We should be able to change that with a few more commands.

Altering the N.P.C.'s Appearance

I believe that the N.P.C. that you just created should be selected, but if any commands that you run to modify it show up on another N.P.C. or do not work and show up with a message similar to "No N.P.C. selected," look at the N.P.C. you created and type "/npc select. This should select the N.P.C.

To have the N.P.C. stare or look at players who approach it, type "/npc lookclose". It should now begin to look at you.

To alter the mob type of the N.P.C. (from a player to something else if you have not changed its mob type yet), type "/npc type [mob-type]". An example of this would be "/npc type pig."

To alter the skin of the N.P.C., what I have been using is "/npc skin [username]". An example of this would be "/npc skin Technoblade," which would alter the N.P.C.'s skin to the skin of the account with the username Technoblade.

Adding Dialogue (or Monologue) to N.P.C.'s

Working with the N.P.C. text editor is able to be a bit tricky, from my experience (you may not experience that, though). You will open up the N.P.C. text editor by typing "/npc text". A message in chat should notify you that you have opened the N.P.C. text editor. You should be provided with a few options on what you would like to do, but I will explain them here just for the sake of clarity.

Waypoints and N.P.C. Movement

Waypoints will allow your N.P.C. to walk around and give more life to it in general. There is also a video showcasing how this works.

How Waypoints Work

Waypoints, as you may have assumed, are what they sound like. They are points you set for the N.P.C. of your choice to walk to. Waypoint commands do not begin with the usual "/npc" command but instead being with "/waypoint".

There are three ways for an N.P.C. to walk, and are triggered or set by using the "/waypoint provider [walk-type]" command. These three walk types are linear, guided, and wander. If you would like to have the N.P.C. wander all around the map, set the walk type to "wander." If you would like to have a set path for the N.P.C., use "linear" (which is the default, I believe) or "guided" and add some waypoints. I feel like "linear" may be easier to work with (though I have not used "guided" before) because the N.P.C. walks in the sequence of waypoints you set, whereas "guided" will walk to the waypoints you set for it randomly.

Creating Waypoints

Waypoints are created using "/waypoint add [x] [y] [z]". Once you set that waypoint, the N.P.C. you have set that waypoint for should immediately begin walking towards that waypoint.

Crouching and Other Things

You are able to have an N.P.C. crouch with "/npc crouch" or "/npc sneak". There are other options as well which I would recommend you view on the Waypoint Wiki page or the Waypoint commands on the Commands page.

Commands List

I apologize for any confusion I may have caused from my not-so-well-worded step-by-step section, so here is a list of commands that I have used that have helped me in N.P.C. customization. Things that you should change will be in brackets.

Useful Commands
Command Description Example
/npc create [name] Creates an N.P.C. The N.P.C. should spawn as an Alex skin or the
skin of the player with the name of your N.P.C.		 /npc create Jeff
/npc lookclose Causes the N.P.C. to look at the nearest player. /npc lookclose
/npc rename [name] Renames the N.P.C. /npc rename Gerald
/npc text Opens and exits the N.P.C. text editor /npc text
/npc select Selects the N.P.C. that the player is looking at.
All commands following that will apply to that N.P.C.		 /npc select
/npc crouch or /npc sneak Causes the selected N.P.C. to crouch/sneak /npc crouch or /npc sneak
/waypoint provider [provider-type] Allows you to alter the way in which the N.P.C. interacts with waypoints or moves /waypoint provider wander
/npc copy Duplicates the selected N.P.C. to your location /npc copy
